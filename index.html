Installing the Seren addon

Current Repository Version: 2.0.1

Instructions for adding this repo:

  • Go to the Kodi file manager
  • Click on "Add source"
  • The path for the source is https://nixgates.github.io/packages
  • Name it "Nixgates" without the quotes
  • Head to " My Addons"
  • Select install an addon from zipfile.
  • When it asks for the location, select "Nixgates", and install repository.nixgates-2.0.1.zip.
  • Go back to My Addons and select "Install from repository"
  • Select the "Nixgates" menu item
  • Open "Video add-ons" and install Seren
  • (Optional) It's suggested to also install the context menu addon

Enjoy!
